Unsupported Functions

The functions listed below are unsupported and are retained only to maintain backward compatibility with previous releases:

 dbConnectObject
dbDisconnectObject
dbIsObjectConnected
dbObjectHasConnection
dbCompactCellView

These functions should not be used in any new scripts or code and should be removed from existing scripts and code where possible.
